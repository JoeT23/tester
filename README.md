# User Guide – GB/UK National Grid Converter

## Introduction

The UK National Grid Converter allows users to convert between UK National Grid coordinates and geographic coordinates (Latitude and Longitude). The application also displays the converted location on an interactive map and allows the most recent conversion to be exported as a CSV file.

### Home Screen
When the application starts, the Home tab is displayed. This page provides a brief description of the application and confirms that it supports: 
- GB/UK National Grid coordinate conversion
- Interactive map display
- CSV export functionality

### Converter Tab
The Converter tab is where all coordinate conversions are performed.

Step 1: Select the Input Type
- Choose one of the two available input formats: Easting / Northing; Grid Reference

Step 2: Enter Coordinates
- Option A – Easting / Northing
Enter:
Easting
Northing
Example: 550000; 105000

- Option B – Grid Reference
Enter a valid UK Grid Reference.
Example:
TQ300800

Step 3: Convert
- Click the Convert button.
- The application will: Validate the input; Convert the coordinates; Display the latitude and longitude; Display the location on an interactive map.
- Viewing Results: The Results box displays the converted coordinates.
- If invalid information is entered, an error message will be displayed instead. Examples include: Missing values; Invalid grid reference; Incorrect coordinate format; When an error occurs, no map is generated.


### Interactive Map
After a successful conversion, an interactive map appears below the results. The map: Marks the converted location; Allows users to zoom in and out; Helps verify the position visually.


### Export CSV Tab
The Export CSV tab allows users to save the most recent successful conversion.

Step 1: Complete a Conversion
- A conversion must be performed first.
- If no conversion has been completed, there is no data available to export.

Step 2: Export
- Click Export Last Result.
- The application creates a CSV file containing the latest conversion.


Step 3: Download
- A download link appears.
- Click the file to save it to your computer and view results.
